# EDAS 系统演示稿

> 演示时间：约 10-15 分钟  
> 演示版本：WorldMonitor · EDAS 集成模块 v2.0

---

## 一、开场白（约 1 分钟）

各位好，今天主要演示 **WorldMonitor 全球态势感知系统** 中关于 **EDAS 事件数据分析** 的集成功能。

EDAS（Event Detection and Analysis System）是我们的**事件检测与分析子系统**，它能够从海量社交媒体数据中自动提取、分类和分析全球范围内的安全事件。目前系统集成了三个数据源：

- **🇭🇰 香港（2019 年）** — 反送中运动抗议事件
- **🇮🇷 伊朗（2019 年）** — 全国性抗议浪潮
- **🇺🇦 乌克兰（2022-2023 年）** — 俄乌战争期间约 36,000 条推文级事件

下面我将从 **地图视角** 和 **深度分析视角** 两个维度来展示系统能力。

---

## 二、地图集成展示（约 3 分钟）

> **操作**: 打开地图，定位到香港/伊朗/乌克兰区域，点击 EDAS 事件点

### 2.1 地图弹出层

**讲解要点**：
- EDAS 事件以 **紫色主题** 在地图上呈现，区别于其他数据源
- 点击事件点 → 弹出详情卡片：显示事件摘要、地区、突发性标记
- 每个 EDAS 弹出层都包含 **「🔍 深度分析」** 按钮

> **演示话术**：
> 
> "大家可以看到，地图上紫色的标记就是 EDAS 事件。和普通抗议事件不同，EDAS 事件带有更丰富的元数据。点击任何一个事件点，会弹出信息卡片，这里显示了事件摘要和地区标识。特别的是，每个卡片底部都有一个「深度分析」按钮——我们稍后会详细演示。"

### 2.2 聚类视图

- 密集区域自动聚类，显示 EDAS 事件计数
- 放大后逐一点位展开

### 2.3 图层自动路由

**讲解要点**：
- EDAS 事件**不全是抗议**——系统自动分类
- 检测关键词 → 分配到 protest / conflict / natural 等图层
- `_category` / `_layer` 标签机制

> **演示话术**：
> 
> "EDAS 的事件并不局限于单一图层。系统通过关键词分析自动识别事件类型——比如包含 'clash'、'military' 的归入冲突图层，包含 'earthquake'、'flood' 的归入自然灾害图层。目前所有 EDAS 事件以紫色风格统一展示，并带有正确的事件类型标签。"

## 三、EDAS 深度分析面板（约 5 分钟）

> **操作**: 点击地图上的「深度分析」按钮，或从面板列表打开 EDAS 分析

### 3.1 事件列表

- 按时间倒序排列的 80 条事件
- 每行显示：地区标签（🇭🇰/🇮🇷/🇺🇦）、突发性标记、摘要片段、关键词标签
- 点击行 → 进入详情

> **演示话术**：
> 
> "左侧打开 EDAS 事件分析面板，这里列出了系统加载的所有事件，按时间倒序排列。大家可以看到香港、伊朗、乌克兰三地的数据混合在一起，每条事件都有地区标识和突发性标记。点击任意事件进入详细分析页。"

### 3.2 事件详情页

打开的详情页包含以下模块，逐一演示：

#### 📝 事件摘要
- 完整的事件描述文本

#### 📊 维度分析 — 关键词权重
- **左侧**：条形图展示 Top 12 关键词权重
- **右侧**：词云可视化，字体大小与权重成正比

> **演示话术**：
> 
> "详情页的核心是关键词维度分析。左侧是关键词权重条形图——颜色越深代表权重越高；右侧是词云，词语越大说明在该事件中越突出。这对快速理解事件主题非常有用。"

### 3.3 🔗 知识图谱 (KG)

**讲解要点**：
- 全局知识图谱数据集（不随当前事件改变）
- 支持文件：政治人物链 (`chain1.json`, `chain_kg_1.json`)、卫星任务链 (`chain_kg.json`)
- 悬停节点高亮相邻关系
- 连线标注关系类型（President、Vicepresident 等）
- 大图模式提供「显示/隐藏标签」切换

> **演示话术**：
> 
> "接下来是知识图谱部分。这里加载的是全局知识数据集——比如美国政治人物权力链（从特朗普到佩洛西的继承关系）、NOAA 卫星任务时序链等。点击文件按钮即可加载对应的力导向图。
> 
> 注意节点间的连线上标注了关系类型——Vicepresident、Successor、Previousmission 等。悬停节点可以高亮其所有邻居，方便探索网络结构。大图模式下右上角有标签切换按钮。"

### 3.4 👥 人物关系链

- GEXF 格式的人物网络数据（`people.gexf` ~ `people4.gexf`）
- 76-121 节点不等，涵盖全球政要关系
- 支持拖拽、缩放、悬停高亮

### 3.5 ⛓️ 时间因果链（核心亮点）

**讲解要点**：
- 基于**三种检测策略**构建因果关联：
  1. **因果语言匹配**（"led to"、"导致"、"triggered"）
  2. **关键词重叠**（共享高频词）
  3. **时间邻近**（同一地区短时间内的连续事件）
- 水平时间轴可视化
- 弧线表示因果关系，颜色深浅表示置信度
- 🖱️ **悬浮卡片**：显示完整事件摘要 + 日期 + 地区 + 突发标记

> **演示话术**：
> 
> "时间因果链是 EDAS 分析的核心特色。系统基于三种策略自动检测事件间的因果关系：
> 
> 第一，因果语言匹配——如果事件摘要中出现 'led to'、'triggered'、'导致' 等词语；
> 第二，关键词重叠——两个事件共享大量关键词；
> 第三，时间邻近——同一地区短时间内连续发生的事件。
> 
> 可视化层面，水平时间轴展示事件顺序，弧线连接因果关系——红色弧线表示高置信度，紫色中等，蓝色较低。**鼠标悬浮在任何节点上**，会弹出精美的信息卡片，显示完整摘要和元数据；悬浮在弧线上则显示因果类型和时间间隔。"

### 3.6 全屏 Overlay 视图

- 通过地图「深度分析」按钮触发
- 右侧全屏滑入面板
- 包含所有上述分析模块
- 支持 ESC 关闭

## 四、新闻点位绘制（约 3 分钟）

> **操作**: 打开「📰 新闻点位绘制」面板

### 4.1 功能介绍

**讲解要点**：
- 输入新闻文本或上传文件 → 自动提取地名 → 在地图上标记黄色光点（含脉冲动画）
- 支持 50+ 城市坐标（乌克兰、俄罗斯、香港、伊朗、全球）
- 支持格式：纯文本、.txt、.json、.jsonl

### 4.2 点位样式说明

**颜色与可见性**：
- 🟡 **黄色圆点** (`medium` 威胁级别) — RGB `[234,179,8]`，基础透明度 63%
- 自动附带 `timestamp` 时间戳 → 触发 **30 秒脉冲动画**（呼吸光晕效果）
- 透明度随地图缩放动态调节：近景 >62% 可见，全球视图约 25%
- 无需开启任何图层开关，绘制后自动显示

> 💡 *之前点位不可见的原因：之前使用 `info` 威胁级别（蓝色），基础透明度仅 31%，全球视图下仅 **12.5%**，几乎透明。现已修复为 `medium` 级别并增加了时间戳触发脉冲动画。*

### 4.3 现场演示

> **演示话术**：
> 
> "这是新闻点位绘制功能。假设我收到一份情报文本——我可以直接粘贴到输入框，或者点击示例文本快速体验。比如点击这个乌克兰示例：
> 
> *'Heavy fighting reported in Bakhmut and Avdiivka. Russian troops advance near Kherson.'*
> 
> 系统会自动提取 'Bakhmut'、'Avdiivka'、'Kherson' 这三个地名，在地图上以 **黄色脉冲光点** 标记。每条新闻还会自动分类事件类型（冲突、抗议、自然灾害等）。"

- 批量提取所有地点并绘制

## 五、技术架构亮点（约 2 分钟）

可根据观众兴趣选择性讲解：

### 数据流
```
EDAS 原始数据 → seed-edas.mjs 种子脚本 → Redis 缓存 ↔ API 端点 → 前端渲染
```

### 前端组件架构
```
data-loader.ts (EDAS 层路由)
    ├── MapPopup.ts (地图弹出层 · 紫色主题)
    ├── EDASDemoPanel.ts (深度分析面板)
    │       ├── D3 力导向图 (知识图谱)
    │       ├── D3 因果链时间轴 (因果可视化)
    │       └── SVG 条形图 + 词云
    └── NewsPlotPanel.ts (新闻点位绘制)
            └── news-plotter.ts (地名提取引擎)
```

### 因果检测算法
```
causal-chain.ts
  ├── detectCausality()       — 因果语言正则匹配 (中/英)
  ├── keywordOverlap()        — Jaccard 关键词重叠
  └── temporalProximity()     — 时间窗口邻近检测
```

### 数据来源
| 源 | 规模 | 格式 |
|---|---|---|
| 香港抗议事件 | ~100 条 | JSON |
| 伊朗抗议事件 | ~80 条 | JSON |
| 乌克兰推文 | ~36,000 条 | JSONL |
| 知识图谱文件 | 10 个文件 | JSON / GEXF |
